This week at CodeCougars, we were given a project prompt. With some assistance from IBM's AI-enhanced development IDE, we got the opportunity to develop a website that would respond to our respective prompts with some utility.

Prompt: Create an AI College Matchmaker with the assistance of IBM Bob
--

For this project, I used the React Native Javascript framework with Node.js. This website is hosted with Netlify.

APIs and Services I integrated:
- Google's Gemini API: AI-enhanced college recommendations and analyses
- Tesla's Groq API: LLM ensemble in tandem with Gemini to avoid rate limits
- College Scorecard API: Open source college data, with in-depth analysis from the Common Data Sets
- Google's OAuth System for login with Gmail

Future possibilities:
- Integrate with the Common App API to directly interface with a user's college applications and credentials
- Further utilization of the account system to store user portfolio for personalized recommendations and competitive leaderboard systes with other users
